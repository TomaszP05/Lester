### Description
Lester is a social kindness application developed on Flutter/Dart, designed to encourage users to achieve small daily acts of kindness challenges that anyone can do. These challenges can help promote positivity and wellness in usersâ€™ daily lives. Once a user enters the application, a kindness challenge will be displayed, where users have the option to complete it or select another challenge. Additionally, users can write a journal entry daily to write thoughts and feelings, reflecting how a kindness challenge that day brought wellness to their lives, as well as other small features to make the app complete.

## Project Information

### Features
- Daily Kindness Challenges (Sent to users to complete.)
- Kindness Quiz & Mood Log (Checks in on users and logs their mood.)
- Insights Widget (Displays inspirational quotes and weather.)
- Progress & Streak Tracking (Tracks progress, streaks, and shows charts/visualizations.)
- Reminders & Notifications (Push/local notifications for challenge availability, completion, and end-of-day reminders.)
- In-App Journal (Users can write thoughts, feelings, and accomplishments.)
- Mood Tracker (Logs overall daily mood and mood before/after activities.)
- Custom Colour Scheme (Allows selection of app background, text, and ambiance colours.)

5. Run `main.dart`
